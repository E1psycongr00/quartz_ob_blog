## 내용(Content)
### System Call
>[!summary] System Call
>프로그램이 OS 커널이 제공하는 서비스를 이용하고 싶을 때 시스템 콜을 통해 실행

>[!tip] 팁
>시스템 콜이 발생하면 해당 커널 코드가 커널 모드에서 실행됨
### System Call의 종류
- 프로세스/스레드 제어
- 파일 I/O 조작
- 정보 유지
- 장치(Device) 관리
- 프로세스 통신 관련
