## 내용(Content)
### Parameter ? == string | undefined
>[!summary]
>?: string => :string | undefined


```ts
interface Person {
	name: string,
	age: number
	job?: string
}
```


? 마크는 js에서 optional chaining 문법에 쓰인다. 그러나 "타입을 정의할 때는 Optional 타입이다." 라고 정의한다.
